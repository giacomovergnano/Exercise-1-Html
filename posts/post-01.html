This is a small tool that allows you to create a machine generated poster. The user only has control on the range of the fon size, the written content and the amoutnof color layers. The rest is all generated automatically by the machine. The colour composition is based on a neural network javascript library, different coordination are assigned to the R G B values. Inspired by a @hardmaru project on neural network’s art.
